# Frontend — InternTrack UI

Next.js 16 + React 19 frontend with glassmorphism design system.

## Key Features
- Tab-based SPA for Student and Admin portals
- Pink glassmorphism theme (Apple-inspired)
- Recharts data visualization
- Responsive mobile-first design
- Lucide React icons

## Development
```bash
npm install
npm run dev
```

Runs on http://localhost:3000

## Structure
```
app/           -> Next.js App Router pages
components/    -> Reusable tab components
lib/           -> Utilities (API client, types, toast)
public/        -> Static assets
```
