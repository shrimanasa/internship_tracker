# Email Notification Templates

## Template Structure
- Pink gradient header with branding
- Content area with styled tables
- Footer with disclaimer

## Triggers
1. Application Created: Confirmation
2. Interview Scheduled: Details + link
3. Offer Received: Details + deadline
4. Deadline Reminder: Action alerts

## SMTP Config
- Provider: Gmail SMTP (587)
- Auth: App-specific password
- TLS: STARTTLS
- Fallback: Graceful skip
