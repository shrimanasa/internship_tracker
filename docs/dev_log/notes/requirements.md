# Requirements Analysis

## Functional Requirements
1. Student registration and profile management
2. Company and internship listing with search/filter
3. Application tracking with status pipeline
4. Interview scheduling and management
5. Offer recording and comparison
6. Document upload and verification
7. Notifications and reminders
8. Admin dashboard with analytics
9. Skill matching engine
10. Audit logging

## Non-Functional Requirements
- Response time < 500ms for API calls
- Support 500+ concurrent students
- Mobile-responsive frontend
- Secure file uploads (5MB limit)
- JWT authentication with 60min expiry
